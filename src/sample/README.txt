В качестве темы для своей курсовой работы я выбрал интернет-браузер. Браузер GreenCube Написан с помощью встроенного в JavaFX компонента WebView.
У браузера на данном этапе самый базовый функционал и такие вещи как сохранение и чтение куки-файлов пока не реализованы. Говорю пока т.к. есть планы на развитие проекта в дальнейшем.
Компонент WebView в JavaFX все-таки не заменит полноценный браузер, например он не поддерживает воспроизведение видео на Youtube или прямых трансляций на Twitch.
Для того чтобы корректно пользоваться браузером нужно зайти под своим аккаунтом Google, т.к. иначе будет требовать пройти капчу.
Для стандартных целей по типу проверки почты в данный момент браузер отлично подходит, также может являться способом анонимно сидеть в сети).
